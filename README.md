# Verilog Sequence Detector

A robust, synthesizable Verilog implementation of a Finite State Machine (FSM) that detects a specific binary sequence. This project includes the complete RTL design, a comprehensive testbench for verification, and a detailed design report.

## 🚀 Features
* **Design Style:** Mealy Finite State Machine.
* **Target Sequence:** `01111110' 
* **Verification:** Self-checking testbench with edge-case scenarios.

## 📁 Repository Structure
* `/src` - Verilog RTL source code.
* `/tb` - Testbench file with simulation stimulus.
* `/docs` - Comprehensive project report (PDF) including state transition tables and K-maps.
